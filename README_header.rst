===================================================
audiorename - Rename audio files from metadata tags
===================================================

Installation
============

From Github
------------

.. code:: Shell

        git clone git@github.com:Josef-Friedrich/audiorename.git
        cd audiorename
        python setup.py install

From PyPI
----------

.. code:: Shell

        pip install audiorename
        easy_install audiorename
